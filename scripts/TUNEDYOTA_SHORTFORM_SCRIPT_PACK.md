# Tuned Yota — Short-Form Script Pack (VO + Text Overlay)

One vertical asset per script works on **YouTube Shorts, Instagram Reels, TikTok, and Facebook**.
Format: ~25–30s, voiceover over branded text overlays (use the animated template's look).
Tone: classy, honest, trustworthy. **Claims rule:** only provable benefits — never overpromise,
never fabricate numbers. **Brand:** no Stage 2/3, no "MAF tune," turbo tier = "Turbo Performance
Calibration," emissions-intact, no "free" claims, no Kevin Whitman.

Each script gives you: the **target query**, the **VO** (read this), the **on-screen text beats**
(4 scenes), and a **caption + hashtags**. The closing scene is identical every time for brand
consistency: *Tuned Yota · tunedyota.com · (612) 406-7117 · MN·IA·WI·ND·SD·NE.*

## Production notes
- **One asset, four platforms:** export the vertical once; post to Shorts, Reels, TikTok, Facebook.
  No per-platform re-edit needed — only the caption hashtags shift slightly if you want.
- **VO options:** your own voice (most trustworthy) or a clone of your voice via ElevenLabs for
  consistency at volume. Keep music low and tasteful.
- **Cadence:** 2–3 of these per week is very achievable since there's no filming — just VO + the
  text template. Start with #1 and #2 (highest-intent).
- **Feeds the Source of Truth:** drop each finished short on the matching `/answers/[slug]` page too;
  the VO script doubles as the page's front-loaded answer text for AI citation.
- **Disclosure:** your own voice over your own branded text needs no synthetic-media label. If you
  ever use a fully AI-generated *voice that imitates a real person*, disclose per platform rules.
