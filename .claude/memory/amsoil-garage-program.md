Tuned Yota is an authorized AMSOIL dealer (dealer referral / ZO number **30713116**). This program turns the site (and later the [[tunedyota-app-program]] Capacitor app) into a branded AMSOIL storefront + service-intelligence tool for the supported Toyota/Lexus lineup.

**Business model = conversion funnel, not just a store.** Every visitor can buy AMSOIL at retail through a Tuned-Yota-branded experience; the order completes on amsoil.com with `?zo=30713116` attached (30-day device attribution) → commission. The jackpot is converting visitors to **Preferred Customer or Dealer** — once enrolled under the ZO they're permanently associated with Tuned Yota and buy under it forever, hands-off. **Hard constraint:** AMSOIL exposes no ordering/payment API; all checkout happens on amsoil.com. We own the experience; AMSOIL owns the cart (zero inventory/payment liability).

**Price agent LIVE (headless browser beats Cloudflare).** amsoil.com is behind Cloudflare (plain server `fetch` → 403), so the agent uses **Playwright headless Chromium** (`scripts/amsoil/lib/browser-fetch.mjs`): a FRESH browser context per product resets the CF cookie (a reused session gets fingerprinted+blocked); stability flags (`--disable-dev-shm-usage/--disable-gpu/--disable-logging`) stop the network-service crash + quiet the log; each product is isolated in try/catch (errors→HELD, browser relaunches if it dies). All 9 products now carry **real `/p/` URLs + live retail prices** (SS oils $17.99, SEVERE GEAR $24.59/$26.69, Multi-Vehicle ATF $24.19, Ea filters $10.60; the 3 Ea filters share one selector page `/p/amsoil-oil-filter-eaoilfilt/?code=`). **Scheduled**: Windows Task Scheduler job "TunedYota AMSOIL Price Sync" (Sun 3:15 AM) → wrapper `C:\Users\grosh\.tunedyota\run-amsoil-price-sync.cmd` (SAFE: only runs on a CLEAN master, else skips + logs — this repo is actively branch-switched) → `node scripts/amsoil/price-sync.mjs --commit` (guardrailed commit+push).
