Installer home bases (origin for round-trip travel when optimizing the event calendar):

- **Aaron** — Twin Cities, MN (covers MN, IA, Fargo ND, + Eau Claire & Madison WI)
- **Cody** — Sioux Falls, SD (covers SD + Omaha NE)
- **Noah** — Sheboygan, WI (covers rest of WI — Green Bay, Milwaukee)

Why it matters: events are almost always **separate round-trips from home base**, not
direct city-to-city tows. So spacing/geography should be judged as spokes from home, not
as a continuous route. E.g. Aaron's Duluth (~155 mi from TC) and Madison (~270 mi from
TC) being only 7 days apart is fine — he rests at home between two independent day-trips,
not towing ~240 mi corner-to-corner. Cody's proposed Sioux Falls event = zero travel
(home). Noah's Sheboygan/Milwaukee/Green Bay cluster is all <70 mi = minimal windshield
time. Confirms the [[held-branches-ship-checklist]] event calendar is well-optimized.
