Regression test `tests/amsoil-page-schema.test.js` asserts every Product has a valid offer. GOTCHA learned: OG/twitter tags are injected by a LATER `build:seo` step (between `<!-- SEO:OG:START -->` markers), NOT by build-amsoil-pages — so never call `buildAmsoilPages()` bare in a test (strips OG, breaks seo.test.js HEAD_PAGES checks); read on-disk pages instead. Lesson: a GSC "critical" ≠ "warning" — always grep for a `Product` node emitting NONE of the three. See [[magnuson-pricing-integration]], [[amsoil-garage-program]].
