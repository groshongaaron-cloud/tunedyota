tunedyota.com migrated **Wix (www host) → Netlify (non-www, clean URLs)**. As of 2026-07-01 Google had discovered only ~3 URLs on the non-www property — a **discovery + migration problem, not indexing-quality**. Diagnosis + fixes:

- **GSC property (initial state, since changed — see "SA + config switch DONE" below):** was URL-prefix **`https://tunedyota.com/`** (non-www, sees only non-www); no Domain property existed and the `gsc-reader` SA was `siteRestrictedUser` (read-only). **Now (2026-07-01):** a Domain property `sc-domain:tunedyota.com` exists, the SA is `siteFullUser` on it, and the measurement config points at the domain property.
- **Sitemap is already fine** — submitted 2026-06-19, downloaded 2026-06-28, **29 URLs, 0 errors/warnings, 0 indexed**. So Google HAS the sitemap; it just hasn't crawled the pages (new-site latency). robots.txt references it; canonicals are non-www; www→non-www 301 works automatically (Netlify domain redirect). Nothing to re-submit.
- **Migration equity leak (FIXED 2026-07-01, master @ f5ac7c2):** old indexed Wix www URLs (`/service-page/*`, `/booking-calendar/*`, `/book-online`, `/about-5`, …) 301'd www→non-www but then hit **404** (no new equivalent) → equity lost. Added **`site/_redirects`** (53 rules) mapping every old Wix path → its new page: model booking/service slugs → the matching vehicle page (tacoma & sc-tacoma → `/toyota-tacoma-ott-tune`, lexus-gx460/470/550 → `/lexus-gx-ott-tune`, etc.), static pages (`/book-online`,`/services`→funnel; `/about(-5)`→`/team`; `/contact`,`/gift-card`,`/payment-request-page`→`/`), + catch-alls `/service-page/*` & `/booking-calendar/*`→funnel. **Old URL list sourced from the Wayback CDX API** (`web.archive.org/cdx/search/cdx?url=tunedyota.com*`). Verified live: old paths 301 to correct new pages, www 2-hop chain resolves, real pages stay 200. This is the main structural fix — Google recrawls the ~70 old URLs it already indexed → follows 301s → discovers/credits the new pages.
- Internal linking was already made comprehensive (the world-class pass — every page cross-links; see [[search-ai-visibility-program]]).
