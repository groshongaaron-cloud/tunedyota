# Tuned Yota — Meta Ads Creative (A/B copy)

Ready-to-launch Facebook/Instagram ad copy. Pair each concept with a Reel from
`instagram-content-kit.md` as the creative. Send traffic to:
`https://tunedyota.com/find-your-exact-tune?utm_source=meta&utm_medium=paid&utm_campaign=<concept>`
(or the link hub `https://tunedyota.com/links`). Pixel + Lead event already track conversions.

**Targeting (all concepts):** Location MN/IA/WI/ND/SD/NE (or radius around event cities).
Interests: Toyota Tacoma/4Runner/Tundra/Land Cruiser, Lexus GX, overlanding, off-roading,
vehicle modification, car tuning, Magnuson Superchargers. Age 25–55, broad gender.
Optimize for **Leads** (the Find Your Exact Tune submit). Start $10–20/day, 5–7 day tests.
