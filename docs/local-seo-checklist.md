## 1. Google Search Console + Bing (do first — gets you indexed)
- [ ] Add `tunedyota.com` in Google Search Console (https://search.google.com/search-console). Verify via DNS TXT (your DNS is at Wix) or the HTML-file method.
- [ ] Submit `https://tunedyota.com/sitemap.xml`.
- [ ] Use "URL Inspection" → Request Indexing on: home, /find-your-exact-tune, /ott-tune, /supercharger, and 2–3 top vehicle pages.
- [ ] Repeat in Bing Webmaster Tools (https://www.bing.com/webmasters) — you can import from GSC. Bing also feeds ChatGPT search.

## 2. Google Business Profile (biggest local lever)
- [ ] Create/claim a profile for **Tuned Yota** at https://business.google.com.
- [ ] Set it as a **Service-Area Business** (hide address) — add your service areas (event cities: Twin Cities, Duluth, Rochester, Mankato, Eau Claire, Madison, Green Bay, Milwaukee, Des Moines, Cedar Rapids, Davenport, Fargo, Sioux Falls, Rapid City, Omaha).
- [ ] **Primary category:** "Auto tuning service" (or "Auto repair shop" if unavailable). Add secondary: "Car repair and maintenance service".
- [ ] Fill: phone, website, hours ("by appointment"), services (OTT Tune, custom calibration, Magnuson supercharger sales/install/calibration), and a keyword-rich description.
- [ ] Upload real photos: installs, dyno/5-gas analyzer, trucks before/after, event setups, team headshots. Add new photos monthly.
- [ ] Verify (video verification is common for SABs — have a Tuned Yota branded item + tools ready).
- [ ] Reality check: Google expects a SAB to cluster near a base location. A single profile centered on your MN base with regional service areas is realistic; coverage in far states (NE/SD) builds as you get reviews/photos from those events.

## 3. Reviews (the #1 ongoing driver — local pack + AI answers)
- [ ] Get your GBP "review short link" (g.page/r/…). 
- [ ] After every tune: text/email the customer the link. Ask them to mention **vehicle + city + what improved** (e.g. "4Runner in Omaha — fixed gear hunting").
- [ ] Aim for a steady cadence (a few per month beats a one-time burst).
- [ ] Respond to **every** review (thank, reference the vehicle/service).
- [ ] Once you have ~5+ Google reviews, tell your developer — we'll add `aggregateRating` to the site schema so stars can show in results.

## 4. Authority backlinks (build entity trust)
- [ ] **Magnuson dealer locator** — confirm your listing links to `https://tunedyota.com` (https://magnusonsuperchargers.com/pages/dealer-locator).
- [ ] **Overland Tailor / OTT** — ask to be listed as an authorized installer with a link.
- [ ] **Forums** (real participation, not spam): TacomaWorld, IH8MUD, Tundras.com, 4Runner forums — helpful posts with your site in your signature/profile.
- [ ] **YouTube** — link tunedyota.com in channel + video descriptions (OTT review videos).
- [ ] Local car clubs, event organizers, and any press → ask for a link.

## 5. Other map/citation listings (NAP consistency)
- [ ] Apple Business Connect (Apple Maps), Bing Places, Yelp, Foursquare, Data Axle.
- [ ] Keep NAP identical to the block at the top of this file on every one.

## 6. Measure
- [ ] Connect Google Analytics 4 + keep Search Console open monthly.
- [ ] Track: impressions/clicks per page, which queries you appear for, review count/rating trend.
